Help & Support

Getting Started with SENSE OS

Welcome to SENSE OS! To help you get started, here are a few steps to familiarize yourself with the system and its features:

  • Voice Commands: Enable the voice assistant to control various features of the system.
  • Screen Reader: Turn on the screen reader to hear descriptions of your actions.
  • Braille Support: Connect a Braille display to interact with SENSE OS more efficiently.
  • Adjusting Settings: Customize font size, contrast, and other accessibility features to fit your needs.

Commands

These are the unique commands of our shell

  • SENSE: Used to access our shell.
  • uninstall-sense: Removes the folder structure and commands themselves.
  • space: Show disk space.
  • memory: Shows the free memory.
  • services: Shows the services that are running.
  • cpu: Shows the use of the cpu.
  • find-file name: Returns the path to the file passed as parameter.
  • processes: Displays the running processes created by the user.
  • order-66 pid: Stops the process whose pid coincides with the past.
  • system-run: Sample the time the system has been in operation.
  • active-connections: Shows you the active network connections.
  • processes-memory: Shows you the processes that consume the most memory.
  • processes-cpu: Shows you the most cpu-consuming processes.
  • help-sense: Opens the index.html file where the user's manual is located.
  • bye: Turns off the system.
  • miguel: Display an ASCII artwork.
  • example: Create a folder if it does not exist yet, in the path created during the SENSE installation, create a script and edit it if it does not exist yet, inside it and execute it, finally return the PID of the execution of this file.
  • no-example: Deletes the folder and the internal file created with the command example.

Common Issues & Solutions

Here are some common issues users may encounter and their solutions:

  • Voice Assistant Not Responding: Ensure the microphone is enabled in your settings and try again.
  • Screen Reader Lag: Adjust the speed of the screen reader in the accessibility settings.
  • Connection Issues: Make sure your internet connection is active, or try restarting your device.
